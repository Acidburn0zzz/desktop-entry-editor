Installation
------------

### Dependencies

* [GTK+ 3](http://www.gtk.org) or newer.
* [pygobject3](http://ftp.gnome.org/pub/GNOME/sources/pygobject/3.0/) or newer.
* [pyxdg](http://www.freedesktop.org/wiki/Software/pyxdg)
